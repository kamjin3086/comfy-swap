Plugin Installation

Recommended Clone the plugin repository for easy updates

1 Navigate to your ComfyUI custom_nodes directory:
cd ComfyUI/custom_nodes
2 Clone the repository:
git clone https://github.com/kamjin3086/ComfyUI-ComfySwap.git
3 Restart ComfyUI
To update later, run git pull in the plugin folder.

Download and manually install the plugin

1 Download the plugin ZIP file:
Download from GitHub
2 Extract the ZIP file
3 Rename folder to ComfyUI-ComfySwap and move to ComfyUI/custom_nodes/
4 Restart ComfyUI
